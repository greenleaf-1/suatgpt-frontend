## ⚠️ 重要提示

### 网络要求
**AI功能必须在校园网环境下使用**
- AI服务位于内网（10.22.18.x）
- 无法在公网环境访问
- 如显示离线，请检查校园网连接

### 数据保存
应用会自动选择最佳存储方式：
- ✅ Supabase可用 → 保存到云端
- ✅ Supabase不可用 → 保存到本地浏览器
- ✅ 数据不会丢失

## 🐛 常见问题

### Q: 显示"无法连接到Supabase"
**A:** 正常现象！应用会自动使用本地存储，所有功能不受影响。

### Q: AI不回复
**A:** 
1. 确认在校园网环境
2. 点击右下角🔵AI设置查看状态
3. 查看错误提示信息

### Q: Edge Function 403错误
**A:** 可以忽略！AI功能不依赖Edge Function。

### Q: 数据会丢失吗？
**A:** 不会！会自动保存到云端或本地浏览器。
