## 🎯 三步开始使用

### **第1步：登录教师后台**
1. 点击主页右上角 **"教师入口 →"**
2. 输入演示账号：
   - 用户名：`teacher`
   - 密码：`teacher123`

### **第2步：管理课程**
1. 点击 **"课程管理"** 标签
2. 点击 **"创建课程"** 按钮
3. 填写课程信息并保存
4. **学生端立即看到新课程！**

### **第3步：推送通知**
1. 点击 **"通知推送"** 标签
2. 填写通知标题和内容
3. 点击 **"发送给所有学生"**
4. **所有学生立即收到通知！**
