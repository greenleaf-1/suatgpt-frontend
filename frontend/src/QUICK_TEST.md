## 🐛 如果遇到问题

### 问题1：应用无法启动
**解决方案：**
```bash
# 清除缓存重新安装
rm -rf node_modules
npm install
npm run dev
```

### 问题2：页面空白
**检查：**
1. 打开浏览器控制台（F12）
2. 查看Console标签页的错误信息
3. 查看Network标签页的请求

### 问题3：AI聊天报错
**这是正常的！** 
- 当前AI功能需要连接内网：`http://10.22.18.12:40011`
- 如果不在校园网内，AI功能会显示错误
- 其他功能（学习中心、通知等）使用Mock数据，不受影响

## 🔄 切换到真实后端

### 前提条件
- ✅ Java后端已启动（端口8080）
- ✅ 数据库已创建并初始化

### 修改配置
编辑 `/utils/api-config.ts`：
```typescript
export const API_CONFIG = {
  BASE_URL: 'http://localhost:8080/api',
  USE_MOCK_DATA: false,  // 改为 false
};
```

### 重启前端
```bash
# 按 Ctrl+C 停止当前服务
# 重新启动
npm run dev
```

### 测试真实后端
1. 使用后端init.sql中的账号登录
2. 验证所有功能
3. 检查数据是否来自真实数据库
